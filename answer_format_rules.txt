ANSWER FORMAT RULES

Keep answers concise and mobile-readable.
Use bullets when they make the answer easier to scan.
Mention only decision points relevant to the question.
Do not list every pathway unless explicitly requested.

SOURCE FORMAT

The source must appear exactly once, as the final line of the answer.
Use this exact format:
Source: <SHORT_SOURCE_LABEL>

Never show file paths.
Never write "Source file".
Never write protocol paths.

MARKDOWN

Do not use markdown bold unless explicitly requested.
Prefer plain text.
