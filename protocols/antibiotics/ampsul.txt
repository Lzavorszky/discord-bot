## INFO_BLOCKS

### indication

This protocol is intended for severe, proven multidrug-resistant or carbapenem-resistant Acinetobacter infection.
Typical use is severe ICU infection, ventilator-associated pneumonia, nosocomial pneumonia, or bloodstream infection.
It is generally used as combination therapy with another active agent according to susceptibility.

Do not use this block to select a dose; use SELECTION_RULES and SELECTED_OUTPUTS.

### microbiology

Sulbactam activity is considered favourable when the sulbactam inhibition zone is 11 mm or greater.
If the inhibition zone is below this threshold or unknown, infectious diseases / microbiology consultation is advised before relying on sulbactam.
Conventional susceptibility testing remains definitive.

### combination_therapy

Use with another active agent according to susceptibility.
Examples of possible active partner agents include colistin or tigecycline.
Avoid monotherapy in severe infection.

### administration

Continuous infusion is preferred for high-dose treatment when feasible.
For CRRT or GFR >=60, dissolve the total daily dose in 250 mL NaCl 0.9% and use pump rate 11 mL/h.
For lower renal tiers, administration and pump rate may need local individualization.

### monitoring

Monitor renal function and clinical response.
Monitor microbiological response when cultures are available.
Escalate to infectious diseases / microbiology consultation if persistent bacteremia, worsening shock, unclear susceptibility pattern, treatment failure, severe renal dysfunction, or unclear colonization vs infection.

## RESTRICTED_OUTPUTS

Never provide:
- ampicillin/sulbactam doses outside SELECTED_OUTPUTS
- treatment advice for non-Acinetobacter infections from this protocol
- monotherapy advice for severe MACI/CRAB infection
- susceptibility interpretation beyond the inhibition-zone statements in INFO_BLOCKS

For info requests:
- answer only from INFO_BLOCKS
- do not add extra doses unless the user explicitly asks for dosing

## SAFETY_RULES

- Select dosing only from renal/RRT status.
- MACI/CRAB severity, inhibition zone, and combination therapy requirements are handled by DEFAULT_FOOTER and INFO_BLOCKS, not by SELECTION_RULES.
- If renal data conflict, IHD has highest priority, then CRRT, then numeric GFR.
- If no renal/RRT information is supplied, give DEFAULT_ANSWER only.
