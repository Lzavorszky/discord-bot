Vancomycin gyors terkep

Tolto dozist kovetoen folyamatos infuzioban adagoljuk.

Folyamatos infuzio, kezdo dozis az elso 24 oraban:

### EN

Vancomycin quick map

Give a loading dose, then dose by continuous infusion.

Loading dose is not affected by renal function:
- TBW <65 kg: 1.5 g
- TBW >=65 kg: 2 g

Continuous infusion, starting dose in the first 24h:
- GFR >=50: 2 g/24h
- GFR 20-49: 1.5 g/24h
- GFR 10-19: 1 g/24h
- CRRT: 2 g/24h
- IHD: 500 mg bolus after HD

For a more exact answer send: body weight in kg and GFR/CRRT/IHD. For TDM adjustment send the vancomycin level.

## INFO_BLOCKS

### administration

Give a loading dose first, then administer vancomycin as continuous infusion.
The loading dose is based on total body weight and is not affected by renal function.
The continuous infusion starting dose is primarily based on renal function or renal replacement therapy status.

### mic_limitation

The dosing table applies to bacteria with MIC 1.0 or below.
If MIC is above 1.0, the source says vancomycin cannot be dosed safely and effectively, and an alternative agent should be chosen.
Do not name an alternative agent from this protocol.

### target_and_monitoring

Target concentration: 20-25 ug/ml.
Check vancomycin level daily with morning labs.
Adjust dose according to measured level using deterministic TDM rules when a level is supplied.

### tdm_adjustment

Vancomycin level adjustment table from the source:
- <10 ug/L: check sampling and administration
- 11-20 ug/L: increase by 500 mg/24h
- 21-25 ug/L: no change
- 26-29 ug/L: decrease by 500 mg/24h
- 30+ ug/L: decrease by 1000 mg/24h

Do not select an adjustment from this block; use SELECTION_RULES and SELECTED_OUTPUTS when a level is supplied.

## RESTRICTED_OUTPUTS

Never provide:
- vancomycin dosing outside SELECTED_OUTPUTS
- a dosing recommendation for MIC >1.0
- an alternative antimicrobial choice from this protocol
- toxicity management not present in INFO_BLOCKS
- a TDM adjustment outside the listed vancomycin level ranges

For info requests:
- answer only from INFO_BLOCKS
- do not add dosing unless the user explicitly asks for dosing or supplies a deterministic selection slot

## SAFETY_RULES

- First-dose vancomycin selection and TDM adjustment are separate deterministic pathways.
- If a vancomycin level is supplied, select the TDM adjustment output and do not also select a first-dose renal tier.
- If only body weight is supplied, provide loading dose and ask for GFR/CRRT/IHD for continuous infusion starting dose.
- If only renal/RRT status is supplied, provide the renal continuous infusion starting dose and the generic loading-dose rule.
- If both body weight and renal/RRT status are supplied, provide both exact loading dose and continuous infusion starting dose.
- IHD has priority over CRRT and numeric GFR if conflicting renal data are supplied.
- CRRT has priority over numeric GFR if both are supplied.
- The source does not specify GFR <10 without IHD/CRRT.
- MIC >1.0 is a safety boundary: do not dose vancomycin from this protocol.
