## ANSWER_POLICY

Can default interpretation be given if pathogen information is missing?
- no

Do NOT provide organism-specific interpretation before pathogen information is available.

If pathogen/result not specified:
- ask which organism/pathogen/resistance marker was detected.

If partial BioFire information exists:
- provide the matching interpretation first
- then ask for missing preferred information (e.g. resistance markers).

## PREFERRED_INFORMATION

Helpful for interpretation and escalation:
- Any resistance genes positive?
- Polymicrobial infection present?
- Intubated/mechanically ventilated?
- CAP vs VAP?
- Shock/hemodynamic instability?
- Prior microbiology available?
- Colonization vs confirmed infection suspected?

Resistance markers include:
- CTX-M
- IMP
- KPC
- NDM
- VIM
- OXA-48-like
- mecA/C
- MREJ

If resistance information missing:
- provide organism-based interpretation first if possible
- then ask for resistance markers when relevant.

## MODIFIER_INFORMATION

These may modify therapy/escalation:
- CTX-M positivity
- carbapenemase genes
- mecA/C or MREJ positivity
- polymicrobial infection
- VAP/mechanical ventilation
- septic shock
- colonization vs infection
- prior resistant microbiology

## PATHWAY_PRIORITY

Priority order (first match wins):

1. Carbapenemase gene (IMP/KPC/NDM/VIM/OXA-48)  -> RESISTANCE_GENES (urgent ID consult)
2. mecA/C or MREJ detected (MRSA suspected)       -> RESISTANCE_GENES
3. CTX-M detected (ESBL suspected)                -> RESISTANCE_GENES
4. Polymicrobial infection                         -> POLYMICROBIAL_ESCALATION
5. Single bacterial pathogen                       -> ORGANISM_TIER_MAPPING
6. Atypical pathogen only                          -> ATYPICAL_PATHOGENS
7. Viral pathogen only                             -> VIRAL_PATHOGENS

## TREATMENT_PATHWAYS

### SPECTRUM_HIERARCHY

The protocol table is organised from left to right by increasing antimicrobial spectrum.

#### Spectrum tiers

| Tier | Antimicrobial |
|---|---|
| 1 | ceftriaxone |
| 2 | cefepime |
| 3 | ertapenem |
| 4 | meropenem + colistin |

### POLYMICROBIAL_ESCALATION

#### Core rule

In polymicrobial infections:
- select therapy corresponding to the HIGHEST antimicrobial tier detected
- therapy must cover ALL detected organisms

#### Escalation logic

Examples:

- Streptococcus pneumoniae + Haemophilus influenzae
  -> Tier 1 -> ceftriaxone

- Streptococcus pneumoniae + Pseudomonas aeruginosa
  -> Tier 2 -> cefepime

- Escherichia coli + CTX-M positive
  -> Tier 3 -> ertapenem

- Pseudomonas aeruginosa + CTX-M positive Enterobacterales
  -> conflicting broad-spectrum requirements
  -> recommend infectious diseases consultation

- Acinetobacter calcoaceticus-baumannii complex
  -> Tier 4 -> meropenem + colistin

#### Important behaviour

When multiple organisms are detected:
- explicitly state polymicrobial infection
- explain escalation rationale briefly
- recommend ID consultation but still provide recommendation

### RESISTANCE_GENES

#### CTX-M

Interpretation:
- probable ESBL-producing organism

Escalation:
- upgrade to Tier 3
- recommended therapy: ertapenem

Suggested wording:
CTX-M positivity suggests ESBL production. Escalation to ertapenem is recommended.

#### Carbapenemases

Includes: IMP, KPC, NDM, VIM, OXA-48-like

Action:
- urgent infectious diseases consultation
- until ID consult available: meropenem + colistin

Suggested wording:
Carbapenemase gene detected. Difficult to treat organism. Advised antibiotic may be insufficient. Immediate infectious diseases consultation is recommended.

#### mecA_C and MREJ

Interpretation:
- MRSA likely

Recommended therapy:
- vancomycin

Special rule:
- if Staphylococcus aureus is the only pathogen: vancomycin monotherapy is sufficient

Suggested wording:
mecA/C & MREJ positivity suggests MRSA. Start vancomycin.

### ATYPICAL_PATHOGENS

Includes:
- Legionella pneumophila
- Mycoplasma pneumoniae
- Chlamydia pneumoniae

Recommended therapy:
- clarithromycin

Rules:
- if ONLY atypical pathogen present: clarithromycin monotherapy
- if bacterial coinfection present: ADD clarithromycin to bacterial coverage

Suggested wording:
Atypical pathogen detected. Add clarithromycin to the regimen.

### VIRAL_PATHOGENS

#### Influenza A/B

Recommended therapy:
- oseltamivir 75 mg BID

#### Other respiratory viruses

Includes: Adenovirus, Coronavirus, Parainfluenza virus, RSV, Rhinovirus/Enterovirus, Human metapneumovirus, MERS coronavirus

Management:
- supportive therapy
- antibacterial therapy generally not indicated unless bacterial coinfection suspected

Suggested wording:
Viral pathogen only detected. Supportive therapy is recommended. Antibacterial therapy is not routinely indicated.

### DOSING_PRINCIPLES

In septic patients during the first 24–48 hours:
- use full-dose antibiotics
- do not reduce dose solely because of renal dysfunction

## SAFETY_NOTES

### VAP_LIMITATIONS

Routine use in ventilator-associated pneumonia is generally discouraged because:
- many nosocomial pathogens are absent from the panel
- resistant ICU flora incompletely represented
- Stenotrophomonas is not included
- positive respiratory PCR may represent colonization
- prior microbiology data is often already available

The panel may still be used for targeted diagnostic questions.

When discussing VAP:
- explicitly mention limitations
- avoid overconfident interpretation
