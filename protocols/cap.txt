## DEFAULT_ANSWER

### HU

CAP gyors terkep
- intubalt / kritikus: BioFire PN trachea/BAL mintabol; eredmenyig ceftriaxone
- hospitalizalt, nem intubalt, nosocomialis riziko nelkul: ceftriaxone + clarithromycin
- hospitalizalt, nem intubalt, nosocomialis rizikoval: levofloxacin
- hazaengedheto, viral negativ: amoxicillin
- hazaengedheto, atypusos gyanu: azithromycin
- influenza: oseltamivir
- aspiratio: antibiotikum csak masodlagos infekcio gyanunall; ha kell: ceftriaxone
- COPD exacerbation bacterial superinfection gyanunall: ceftriaxone + clarithromycin

Pontosabb valasztashoz kuldd: hazaengedheto / hospitalizalt / intubalt, illetve viral teszt, atypusos gyanu, nosocomialis riziko.

### EN

CAP quick map
- intubated / critical: BioFire PN from trachea/BAL sample; ceftriaxone until result
- hospitalized, not intubated, no nosocomial risk: ceftriaxone + clarithromycin
- hospitalized, not intubated, nosocomial risk: levofloxacin
- dischargeable, viral negative: amoxicillin
- dischargeable, atypical suspicion: azithromycin
- influenza: oseltamivir
- aspiration: antibiotics only if secondary infection suspected; if needed: ceftriaxone
- COPD exacerbation with suspected bacterial superinfection: ceftriaxone + clarithromycin

For more exact selection send: dischargeable / hospitalized / intubated, plus viral test, atypical suspicion, nosocomial risk.

## INFO_BLOCKS

### diagnostic_testing

Outpatient during epidemic period:
- combined antigen test indicated during epidemic period
- GeneXpert four-panel not indicated
- BioFire RP2.1 not indicated
- urinary antigen if clinically suspected
- BioFire PN not indicated

Hospitalized during epidemic period:
- combined antigen test if clinical suspicion and negative rapid test
- GeneXpert four-panel if pertussis suspicion
- BioFire RP2.1 if atypical pathogen suspicion
- urinary antigen indicated
- BioFire PN not indicated

Intubated during epidemic period:
- combined antigen test not indicated
- GeneXpert four-panel if pertussis suspicion
- BioFire RP2.1 indicated
- urinary antigen indicated
- BioFire PN indicated; order immediately

### aspiration

Aspiration event alone does not automatically require antibiotics.
Antibiotics are used only if secondary infection is suspected.
If antibiotic is needed, the pathway names ceftriaxone.
CAP does not provide ceftriaxone dosing.

### viral_and_influenza

If viral test is positive in a dischargeable patient, antibiotics are not required unless bacterial superinfection is suspected.
Influenza treatment named in this protocol is oseltamivir.
CAP does not provide oseltamivir dosing.

### allergy_and_safety

Ask about allergy history.
Ask about renal function when antibiotic dosing is requested because dosing belongs to linked drug protocols.
Ask about shock / hemodynamic instability when clinically relevant.
Escalate severe or unclear cases to senior clinician / ID specialist.
No patient identifiers.

### routing_todos

TODO: HAP/VAP routing requires human review. Do not add HAP, VAP, hospital-acquired pneumonia, ventilator-associated pneumonia, or nosocomial pneumonia aliases to CAP unless source content explicitly covers those pathways.

### steroid_note

If shock is present in intubated CAP, the source protocol says to consider corticosteroids: methylprednisolone.
Methylprednisolone is not an antibiotic and is not included in antimicrobial dosing links.
CAP should not provide steroid dosing unless a steroid protocol is explicitly linked.

## RESTRICTED_OUTPUTS

Never provide:
- antibiotic or antiviral doses from CAP
- renal dose adjustment from CAP
- ceftriaxone, clarithromycin, levofloxacin, amoxicillin, azithromycin, or oseltamivir dosing unless a linked protocol is active
- methylprednisolone dosing from CAP
- nosocomial-risk pathway if patient is intubated
- HAP/VAP pathway selection from CAP unless source content is explicitly added
- antibiotics for viral-only dischargeable illness unless bacterial superinfection is suspected

For dosing requests:
- use LINKS
- if the linked target protocol is missing, say the target_missing_behavior sentence

For info requests:
- answer only from INFO_BLOCKS
- do not add dosing

## SAFETY_RULES

- CAP selects treatment pathway only; it does not dose drugs.
- Intubated status has highest priority and overrides nosocomial-risk logic.
- After a pathway names an antimicrobial, keep it as last_selected_item so a follow-up dosing request can activate LINKS.
- If multiple antimicrobials are selected, and the user asks "dose?", ask which one unless the user named one.
- If the user supplies GFR/renal data during a CAP pathway, store it and transfer it through LINKS; do not use it inside CAP.
