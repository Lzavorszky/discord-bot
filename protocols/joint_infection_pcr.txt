## INFO_BLOCKS

### recognition

BioFire JI interpretation only answers when at least one pathogen is supplied.
Resistance markers are interpreted only together with the detected pathogen list.
Pathogen names must be treated as microbiology entities.
Do not route pneumonia panel interpretation from this protocol.

### anaerobe_guidance

Anaerobes: if using cephalosporin therapy, add metronidazole. No additional anaerobe agent is needed with a penicillin derivative such as amoxicillin/clavulanate or piperacillin/tazobactam, or with a carbapenem such as imipenem or meropenem.

### syndrome_context

For skin-soft tissue infections safe to use this protocol as is.
For nosocomial intra-abdominal infection, use JI PCR for regimen broadening only. Do not narrow below meropenem from PCR alone; wait for conventional culture and susceptibility.

### resistance_genes

Resistance markers are BioFire-positive only in the context of detected pathogen(s), so they must not be interpreted alone.
CTX-M suggests ESBL-producing Enterobacterales and upgrades relevant detected gram-negative organisms to meropenem in this protocol.
Carbapenemase genes select meropenem + colistin and require ID consultation.
mecA/C, mecA/B, or MREJ with Staphylococcus aureus suggests MRSA and points to vancomycin.
VanA/B points to linezolid for Enterococcus context.

### limitations

Unlike narrower syndrome PCR tests, JI PCR may be used across multiple syndromes and polymicrobial infections. Review conventional cultures and susceptibilities when available, and contact ID for uncertainty.
After telephone notification, send the sample directly to the microbiology laboratory as with other PCR tests.

## RESTRICTED_OUTPUTS

Never provide:
- antimicrobial doses from BioFire JI
- renal adjustment from BioFire JI
- organism-specific interpretation when no pathogen is supplied
- standalone resistance-marker interpretation without a detected pathogen
- pneumonia panel interpretation from this joint infection panel protocol
- narrower therapy in polymicrobial infection when a broader tier is required

For dosing requests:
- use LINKS
- if the linked target protocol is missing, say the target_missing_behavior sentence

For info requests:
- answer only from INFO_BLOCKS
- do not add antimicrobial dosing

## SAFETY_RULES

- At least one pathogen is required before interpretation.
- Resistance markers are interpreted only with detected pathogen(s); if marker-only input is supplied, ask for the pathogen.
- Bare Klebsiella, Enterococcus, or Streptococcus requires species clarification only when the species changes antimicrobial selection.
- No multi-turn question tree is needed; if missing, return DEFAULT_ANSWER only.
- Keep last_selected_item so a follow-up dosing request can activate LINKS.
- If multiple antimicrobials are selected and the user asks for dose, ask which antimicrobial unless the user named one.
- If GFR/renal data is supplied, store it and transfer it through LINKS; do not use it inside BioFire JI.
