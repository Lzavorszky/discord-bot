## DEFAULT_ANSWER

Return the complete perioperative steroid guide table. Do not ask for surgery type before showing the table.

## ANSWER_POLICY

This protocol is active only for perioperative, preoperative, operation/procedure, or surgery-day steroid questions.

When the user asks about perioperative steroid management, chronic steroid use before surgery, or stress-dose steroids, return the complete steroid guide table. Do not ask for surgery size first; the table is short and the clinician will choose the applicable row.

Do not calculate steroid equivalence beyond listing the source equivalence table.

Do not provide steroid advice for non-perioperative contexts.

## REQUIRED_INFORMATION

Required before answering:
- perioperative/surgery/procedure context, or active perioperative context from the preceding turn

No dose, duration, or surgery-size input is required before returning the table.

## DEFAULT_QUESTION

Are you asking about perioperative steroid management around surgery?

## PATHWAY_PRIORITY

1. If there is no perioperative/surgery/procedure context and no active perioperative context, do not answer from this protocol.
2. If perioperative context is present, return the complete steroid guide table.
3. If the user asks for equivalence, include the equivalence table.
4. Do not ask for surgery size before returning the table.

## SELECTION_RULES

No deterministic selection, calculation, or dose conversion is performed. The table is returned as source guidance.

## INFO_BLOCKS

### steroid_guide_full

Chronic steroid use - perioperative action:

| Regular steroid use | Action |
|---|---|
| methylprednisone < 8 mg | Continue the usual dose; no additional steroid is needed. |
| methylprednisone >= 8 mg for at least 3 months | Usual dose plus hydrocortisone supplementation. |

Hydrocortisone supplementation by surgery size:

| Surgery size | Example from source | Hydrocortisone supplementation |
|---|---|---|
| small surgery | inguinal hernia | 1 x 25 mg hydrocortisone at the start of the operation |
| medium surgery | laparoscopic cholecystectomy | 1 x 25 mg hydrocortisone + 4 x 25 mg for 24 hours |
| major surgery | PPPD | 1 x 25 mg hydrocortisone + 4 x 25 mg for 48 hours |

Steroid equivalence table:

| Steroid | Equivalent dose | Glucocorticoid:mineralocorticoid activity | Duration |
|---|---:|---|---|
| methylprednisone | 8 mg | 5:0.5 | 12-36 h |
| dexamethasone | 1.5 mg | 30:0 | 36-54 h |
| hydrocortisone | 40 mg | 1:1 | 8-12 h |
| prednisolone | 10 mg | 4:0.8 | 12-36 h |
| fludrocortisone | 4 mg | 10:250 | 24 h |

### source_terms

The source uses:
- "Rendszeresen szedett szteroid" for regular/chronic steroid use
- "methylprednison < 8mg"
- "methylprednison 8+ mg legalább 3 hónapig"
- "hydrocortison kiegészítés"
- "kis műtét", "közepes műtét", "nagy műtét"

## RESTRICTED_OUTPUTS

Never provide:
- non-perioperative steroid treatment advice
- steroid equivalence calculations beyond the listed source table
- adrenal insufficiency diagnosis or testing advice unless present in the source
- tapering advice
- pediatric dosing
- pregnancy-specific steroid advice
- hydrocortisone regimens other than the source rows

## SAFETY_RULES

- Return the complete source table for perioperative steroid questions.
- Do not ask for surgery type before returning the table.
- Do not extrapolate to surgery categories not listed in the source.
- If the user asks for an unlisted steroid, provide only the equivalence table if applicable and state that unlisted steroid guidance is not specified.
- If the user asks for conversion/calculation, list the equivalence table but do not calculate a patient-specific converted dose.
