## DEFAULT_ANSWER

Provide height in cm and actual body weight in kg. This calculator intentionally requires cm and kg input.

## SELECTION_RULES

selection_mode: none

Unit rule:
- Accept height only in cm.
- Accept actual body weight only in kg.
- If height or weight is supplied in another unit, ask the user to resend the value in cm and kg.

Calculation rules:
- height_m = height_cm / 100
- BMI = actual_weight_kg / (height_m * height_m)
- BSA_Mosteller_m2 = square_root((height_cm * actual_weight_kg) / 3600)
- IBW_male_kg = 50 + 0.91 * (height_cm - 152.4)
- IBW_female_kg = 45.5 + 0.91 * (height_cm - 152.4)
- Adjusted_body_weight_male_kg = IBW_male_kg + 0.4 * (actual_weight_kg - IBW_male_kg)
- Adjusted_body_weight_female_kg = IBW_female_kg + 0.4 * (actual_weight_kg - IBW_female_kg)

Where sex changes the equation, calculate and report both male and female formula outputs. Do not ask for sex just to calculate this protocol.

## INFO_BLOCKS

### formula_notes

BMI uses weight in kg divided by height in meters squared. BSA uses the Mosteller equation. IBW uses the Devine-style adult equation in cm. Adjusted body weight here means AdjBW40: IBW + 0.4 * (actual body weight - IBW).

### boundaries

These equations are mathematical estimates only. They do not determine which weight scalar should be used for a specific drug, nutrition plan, ventilator setting, renal estimate, or clinical decision unless another active protocol explicitly says so.

## RESTRICTED_OUTPUTS

Never provide:
- medication dosing advice from this calculator alone
- renal function estimation from this calculator alone
- nutrition targets from this calculator alone
- pediatric interpretation
- pregnancy-specific interpretation
- sex-specific clinical advice beyond reporting both formula outputs
- conversion from lb/in to kg/cm; ask for cm and kg instead

## SAFETY_RULES

- Require height in cm and actual body weight in kg before calculating.
- Report both male-formula and female-formula IBW/adjusted-body-weight results.
- If actual body weight is at or below IBW, note that adjusted body weight may not be the appropriate scalar for dosing unless a drug-specific protocol says so.
- Do not infer clinical appropriateness of any body-size scalar.
